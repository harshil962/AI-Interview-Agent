Project Info

About This Project

❌
Problem

Students lack access to realistic, feedback-driven interview practice before entering the job market.

⚙️
Solution

AI-powered mock interviews with instant scoring, plus n8n automation for reminders and result summaries.

✅
Outcome

Better confidence, actionable feedback, and a tracked performance history to guide improvement.

The AI Interview Practice Agent is a learning-based web application developed using Python, Flask, SQLite, Bootstrap, and n8n.

It helps students and freshers practice interview questions in different topics such as Python, DBMS, OOP, and HR. The system evaluates answers, provides feedback, and stores history for progress tracking.

The goal of this project is to improve interview confidence, communication, and technical understanding through consistent, AI-guided practice.

🐍

Python & Flask

Backend powered by Flask with SQLite for fast and lightweight data storage.

🤖

AI Evaluation

Each answer is evaluated by AI which provides a score and detailed constructive feedback.

⚙️

n8n Automation

After every interview, n8n logs your result and can trigger a personalized email feedback summary — no manual effort needed.

📊

Progress Tracking

Full history dashboard showing scores over time, best topics, and areas needing improvement.
